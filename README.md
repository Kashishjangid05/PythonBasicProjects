## 🛠️ Setup and Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Kashishjangid05/PythonBasicProjects.git
   cd PythonBasicProjects
   ```

2. **Run a project**
   Navigate to the directory of the project you want to run. For example:
   ```bash
   cd BasicCalculator
   python main.py
   ```
   *(Note: Script names might vary depending on the project. Check the source files inside each directory)*

3. **Install dependencies**
   Some projects might require external libraries. If a `requirements.txt` file is present in a project's directory, install the required packages using pip:
   ```bash
   pip install -r requirements.txt
   ```
